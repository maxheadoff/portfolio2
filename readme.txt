Приложение для обработки текста.
Производит следующие манипуляции с исходным текстом:
1) удаление слов длиной менее указанного числа символов
2) удаление знаков препинания.

Использование:

TextTtransformer.exe имяФайлаИсточника имяФайлаРезультата МинимальнаяДлинаСлова  [Y(если необходимо убирать знаки препинания)]

Например: TextTtransformer.exe text.txt result.txt 6 y

или

TextTtransformer.exe -d каталогИсходныхФайлов каталогДляРезультатов МинимальнаяДлинаСлова [Y(если необходимо убирать знаки препинания)]

или
TextTtransformer.exe -f ПакетныйФайл.csv

ожидается, что файл внутри содержит строки с параметрами: 
ИмяФайлаИсточника,ИмяФайлаРезультата,МинимальнаяДлинаСлова,УдалятьЛиЗнакиПрепинания

Например:
test.txt,res.txt,3,y
test.txt,res1.txt,5,n
test.txt,res2.txt,2,y

Ограничения:
Обрабатываемый файл должен содержать текст в формате UTF8.
Программа не распознает переносы слов.

Использовалась среда VS2012, Windows 7
Для компиляции проекта необходимо подключить библиотеку NLog.4.3.5 (можно  использовать NuGet).

В рамках проекта не ставилась задача максимального покрытия кода тестами, реализована только демонстрация возможностей.
